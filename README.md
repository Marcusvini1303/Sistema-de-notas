# 📊 Sistema de Notas em Python

Este projeto é um sistema de gerenciamento de notas escolares desenvolvido em Python utilizando Programação Orientada a Objetos.

## Funcionalidades

- Cadastro de alunos
- Cadastro de disciplinas
- Lançamento de notas
- Cálculo de média dos alunos
- Consulta de notas

## Tecnologias utilizadas

- Python
- Programação Orientada a Objetos (POO)

## Estrutura do projeto

main.py → execução do sistema  
aluno.py → classe aluno  
professor.py → classe professor  
disciplina.py → disciplinas  
notas.py → gerenciamento das notas
